# Python Algorithm Study

파이썬을 활용한 알고리즘 문제 풀이, 문법 연구, 그리고 실험을 위한 레포지토리입니다.

## 📂 폴더 구조

### 1. `problems/` (문제 풀이)
실전 코딩 테스트 문제 풀이 코드를 모아두는 곳입니다.
- `baekjoon/`: 백준 온라인 저지
- `programmers/`: 프로그래머스
- `leetcode/`: 리트코드

### 2. `algorithms/` (알고리즘 이론)
자주 사용되는 알고리즘과 자료구조의 구현체 및 템플릿입니다.
- DFS/BFS, 이진 탐색, 다익스트라 등

### 3. `python_study/` (파이썬 탐구)
파이썬 언어 자체에 대한 깊이 있는 학습 기록입니다.
- `syntax/`: 파이썬 문법 심화, 팁
- `libs/`: `itertools`, `collections`, `heapq` 등 코테 필수 라이브러리 활용법

### 4. `playground/` (실험실)
자유롭게 코드를 작성하고 테스트해보는 연습장입니다. (형식에 구애받지 않음)

## 🛠 환경 설정

Conda를 사용하여 환경을 설정합니다.

```bash
conda env create -f environment.yml
conda activate coding_test
```
