# Coding Test Practice

파이썬을 이용한 코딩 테스트 연습 레포지토리입니다.

## 폴더 구조

- `problems/`: 문제 풀이 코드
  - `baekjoon/`: 백준 온라인 저지
  - `programmers/`: 프로그래머스
  - `leetcode/`: 리트코드
- `algorithms/`: 알고리즘 이론 및 템플릿 코드

## 환경 설정

Conda를 사용하여 환경을 설정합니다.

```bash
conda env create -f environment.yml
conda activate coding_test
```
